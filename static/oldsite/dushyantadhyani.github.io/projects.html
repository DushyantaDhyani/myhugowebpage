Event and Participant extraction from Natural Language TextJan-June,2015

As part of the Structuring Story Chains project, I worked on the following.

  • Worked on extracting events and their corresponding participants from News Events as available in the ECB+ Corpus. Used a Conditional random classifier with various structural, sentactic, semantic and latent (Word embedding) features to create near state of the art base models on top of which the remaining framework would be built.
  • Analysed the suitability of the ECB+ Corpus and the TimeML Corpus for the task.
  • Worked on adding Temporal annotation tagger to enable chronological ordering of the events.

MiniOS Sep-Oct,2012

A miniature version of an x86 Operating System consisting of the following components.

  • File System supporting Directories, Sub Directories and Files.
  • A miniature shell supporting basic commands like ls,mkdir,cd,del,execute etc.
  • A Process table with maximum 8 processes.
  • Context Switching between the processes using Round Robin Schedulling.
